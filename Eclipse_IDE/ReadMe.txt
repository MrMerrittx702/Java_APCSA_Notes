This folder is for files associated with the Eclipse IDE.

eclipseConfig.epf
  This is a configuration file that needs to be imported to your eclipse workspace for this class. 
  Go to:
    File > Import
    Select arrow next to General
    Click Preferences then Next
    Click browse and navigate to the eclipseConfig.epf file
    Check import all if it is not already checked
    Click Finished.
    
Other Configuration notes:
  Set up the IDE
  Exit out of any windows present in the editor.
  In the top navigation bar select Window then Show View
  Click Package Explorer and drag it to where you prefer on screen.
  Repeat but this time click Console
  Repeat but this time click Terminal
